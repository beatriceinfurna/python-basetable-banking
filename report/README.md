# Report

This folder contains the **final documentation and visual outputs** of the project.

The material included here summarizes the results of the exploratory data analysis and supports the main findings discussed in the Jupyter notebook and the final report.

---

## Folder Structure

- **final_report.pdf**  
  The final project report.  
  It presents the project objectives, data description, methodological approach, key analyses, results, and conclusions.

- **visualizations/**  
  This subfolder contains all figures generated during the exploratory and descriptive analysis phase.  
  The visualizations are referenced in the report and help illustrate client behavior, transaction patterns, and key financial indicators.

- **README.md**  
  This file provides an overview of the contents of the report folder.

---

## Purpose of the Report

The report aims to:

- Summarize the main insights extracted from the banking dataset  
- Provide an interpretable and visual explanation of client behavior  
- Support analytical conclusions with descriptive statistics and plots  
- Serve as a standalone document complementing the Jupyter notebook analysis  

All analyses are based on **synthetic banking data** used for academic and educational purposes.

---

## How to Use This Folder

- Start with **final_report.pdf** for a complete overview of the project and conclusions  
- Refer to the **visualizations/** folder for detailed graphical support of the analyses  
- Use this material together with the notebook to fully understand the data processing and analytical steps
